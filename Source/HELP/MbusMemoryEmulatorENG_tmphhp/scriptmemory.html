Plc memory
   
To access to the plc memory (simulated), use the following instructions:
Instructions
Designation
GetBitValue(BitNo: Integer):boolean;
Get bit value
SetBitValue(BitNo: Integer; Value: Boolean);
Write a bit
GetRegisterValue(RegNo: Integer):integer;
Get a register value
SetRegisterValue(RegNo: Integer; Value: Integer);
Write a register value
GetRegisterDValue(RegNo: Integer;):Longint;
Get a a register value (double word)
SetRegisterDValue(RegNo: Integer; Value: Longint);
Write a register value (double word)
Log(Text:string);
Write a log text
ScriptSpeed:=speed:integer;
Script speed (0=full speed)
Terminated: boolean;
Script terminated flag by user
