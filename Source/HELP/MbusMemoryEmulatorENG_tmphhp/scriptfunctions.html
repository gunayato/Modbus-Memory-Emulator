Functions
   
Declare a function with the following manner:

function FunctionName(Parameters): ResultType;
var
    Variables definitions;
begin
    ...
end;


Example:

function Mul2(Val: integer): Integer;
begin
  Result := Val * 2; // Result returned by the function = integer
end;

Call the Mul2 function:
Res := Mul2(2); // Res = 4
