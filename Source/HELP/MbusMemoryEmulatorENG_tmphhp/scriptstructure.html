Script structure
   
A script structure looks like:

unit Script; // Script name

// Main procedure will be called in first
function Main: boolean;
begin
     ScriptSpeed := 1; // 0=Full speed

    // Loop that will be executed until stop signal from the user
     while not Terminated do begin
            // ...
            // Write here your own program
            // ...
     end;

     result := True; // Function result
end;

end. // End of the script



Note: '//' means a comment.
