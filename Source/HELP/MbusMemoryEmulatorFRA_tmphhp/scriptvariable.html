Variable
   
La déclaration d'une variable s'effectue de la façon suivante:
var identifierList: type;

Exemple:
var Val: integer;

Exemple dans la fonction Main:
function Main: boolean;
var
   Val1, Val2: integer;
   Float: Single;
begin
   Val1 := 123;
   Val2 := Val1+1;
   Float := 123.65;
   ...
end;
