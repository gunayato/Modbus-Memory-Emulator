Traitement conditionnel
   
La séquence if ... then ... else ... permet d'effectuer un traitement conditionnel et prend plusieurs formes:

Forme 1:
if then instructions;

if J <> 0 then F:=1; // Si J est différent de 0 alors F:=1


Forme 2:
if expression then instructions1 else instructions2;

if J <> 0 then F:=1 else F:=0; // Si J est différent de 0 alors F:=1 sinon F:=0


Forme 3:
if expression then
begin
    instructions1;
    instructions2;
    instructionsN;
end
else if expression then instructions;
