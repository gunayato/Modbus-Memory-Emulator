Mémoire automate
   
Pour accéder à la mémoire (simulée) de l'automate, utilisez les instructions suivantes:
  
Instructions
Désignation
GetBitValue(BitNo: Integer):boolean;
Obtenir la valeur d'un bit
SetBitValue(BitNo: Integer; Value: Boolean);
Ecrire la valeur d'un bit
GetRegisterValue(RegNo: Integer):integer;
Obtenir la valeur d'un registre
SetRegisterValue(RegNo: Integer; Value: Integer);
Ecrire la valeur d'un registre
GetRegisterDValue(RegNo: Integer;):Longint;
Obtenir la valeur d'un registre double
SetRegisterDValue(RegNo: Integer; Value: Longint);
Ecrire la valeur d'un registre double
Log(Text:string);
Ecrire un message dans la fenêtre log
ScriptSpeed:=vitesse:integer;
Vitesse de script (0=pleine vitesse)
Terminated: boolean;
Demande d'arrêt du script par l'utilisateur
