Structure d'un script
   
Un script s'articule (au minimum) de la façon suivante:

unit Script; // Nom du script

// La fonction Main est appelé en premier lors du lancement
// de l'émulateur
function Main: boolean;
begin
     ScriptSpeed := 1; // 0=Pleine vitesse du script

    // Boucle d'execution => le script tourne en permanence tant que l'utilisateur
    // n'a pas stoppé l'émulateur ou qu'aucune erreur n'est apparue
     while not Terminated do begin
            // ...
            // Ecrivez içi votre traitement
            // ...
     end;

     result := True; // Résulat de la fonction (facultatif)
end;

end. // Fin du script



Note: '//' signifie le début d'un commentaire
