Fonctions
   
Une fonction se déclare de la façon suivante:

function NomDeFonction(Paramètres): TypeDeRésultat;
var
    Définitions des variables;
begin
    ...
end;


Exemple:

function Mul2(Val: integer): Integer;
begin
  Result := Val * 2; // Result = résultat renvoyé par la fonction = integer
end;

L'appel de la fonction Mul2 s'effectue comme suit:
Res := Mul2(2); // Res = 4
